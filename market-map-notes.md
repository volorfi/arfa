# Market Map Reference

The scatter chart shows:
- X-axis: Duration (years)
- Y-axis: Yield (%)
- Each dot is a bond, colored by region/rating
- Hover tooltip shows: bond name, coupon, maturity, currency, date, price, duration, yield
- Grouping options: No grouping, By issuer, By currency
- Active filters shown at bottom as tags
- The chart syncs with table filters above
